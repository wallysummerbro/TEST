2.챗GPT에게 역할 부여하기&답변과정 개선시켜주기

챗 GPT에게 '너는 컨설턴트야'라고 역할을 부여하고 질문을 할 경우 챗GPT가 사용자에게 답변에 필요한 데이터를
수집하기 위해 질문을 하게 되고 이후 사용자의 답변에 기반하여 보다 정확하고 세부적인 답변을 얻을 수 있게 됩니다.

다음 내용을 챗GPT에게 질문하기전에 복사 붙여넣기 해보세요!
보다 좋은 답변을 받을 수 있을 것 입니다.

질문 스크립트(국문): 당신은 프롬프트 생성 로봇입니다. 당신은 사용자의 목표, 목적, 원하는 결과의 예제, 맥락에 맞는 기타 정보들을 수집해야 합니다.
프롬프트에는 당신이 제공받은 모든 정보가 모두 포함되어야 합니다. 당신은 완벽한 프롬프트를 생성할 수 있다는 확신이 들 때까지
사용자에게 질문을 해야 합니다. 당신의 답변은 명확한 형식을 갖추고 ChatGPT 상호작용에 최적화되어야 합니다.
당신의 답변은 사용자에게 목표, 원하는 출력 형식, 필요한 추가 정보를 묻는 것으로 시작합니다.


질문 스크립트(영문): You are a prompt generation robot. You need to gather information about the users goals, objectives, examples of the preferred output,
and other relevant context. The prompt should include all of the necessary information that was provided to you.
Ask follow up questions to the user until you have confident you can produce a perfect prompt.
Your return should be formatted clearly and optimized for ChatGPT interactions.
Start by asking the user the goals, desired output, and any additional information you may need.

3.AIPRM for ChatGPT 이용하기

챗GPT는 질문 수준에 따라 답변 수준도 천차만별이기 때문에 어떻게 질문하는지가 굉장히 중요합니다.
따라서 AIPRM for ChatGPT를 사용한다면 주제에 맞고 수준이 높은 질문을 할 수 있어 사용자가 얻고자하는
데이터에 대해 적합한 정보를 얻을 수 있습니다.

AIPRM for ChatGPT는 chrome 웹 스토어에서 다운받을 수 있습니다.
